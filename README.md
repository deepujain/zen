# Zen 🪷 - Spa Management App

Welcome to Zen 🪷, a modern and intuitive management dashboard for spa businesses. This application provides a comprehensive suite of tools to track sales, manage expenses, monitor staff attendance, and generate AI-powered business insights.

## Features

- **Dashboard**: Get an at-a-glance overview of your spa's performance with key metrics like daily, month-to-date, and year-to-date sales, customer counts, and staff attendance.
- **Sales Management**: Record new sales with a detailed form, view sales records with interactive charts, and filter by different time periods (Today, MTD, YTD). Includes visual breakdowns of sales by payment methods, therapists, and therapies.
- **Expense Tracking**: Monitor your business expenses, view total and average daily expenses, and see breakdowns by category.
- **Staff Management**: Maintain a list of staff members, view detailed profiles with their performance summaries, and manage their daily attendance.
- **Attendance Tracking**: A visual attendance system with a calendar view and a drag-and-drop board to update staff status (Present, Late, Absent) for any given day.
- **Data Export**: Export your sales data for any period to a CSV file for offline analysis or record-keeping.
- **Responsive Design**: A fully responsive interface that works seamlessly on both desktop and mobile devices.
- **Theming**: Switch between light and dark modes to suit your preference.

## Tech Stack

This project is built with a modern, production-ready technology stack:

- **Framework**: [Next.js](https://nextjs.org/) (using the App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Generative AI**: [Genkit](https://genkit.ai)
- **State Management**: React Context API with Hooks
- **Forms**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/) for validation
- **Charts**: [Chart.js](https://www.chartjs.org/) with [react-chartjs-2](https://react-chartjs-2.js.org/)
- **Icons**: [Lucide React](https://lucide.dev/guide/packages/lucide-react)

## Getting Started

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

### Prerequisites

- Node.js (v18 or later recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:deepujain/zen.git
   ```
2. Navigate to the project directory:
   ```bash
   cd <project-directory>
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

To run the development server, execute the following command:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

The application uses an in-memory data store for demonstration purposes, so any changes made during a session will be reset upon restarting the server.
